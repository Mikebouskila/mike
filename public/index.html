REST API Documentation

Welcome to the documentation for the REST API of our application. Below, you will find detailed information about available endpoints and how to use them.

Toy API

The Toy API provides endpoints for managing toy-related data.

Get All Toys

Endpoint: GET /toys

Description: Retrieves a list of all toys.

Request:


Method: GET
URL: /toys
Query Parameters:
- parPeg (optional): Number of items per page (default: 10).
- page (optional): Page number (default: 1).
- sort (optional): Sorting field (default: '_id').
- reverse (optional): reverse sorting order (true: reverse = yes).

Response:


Status: 200 OK
Body: [Array of toy objects]

Example:


GET /toys?page=2&parPeg=10&sort=name

Response:
Status: 200 OK
Body: [
    {
        "_id": "1",
        "name": "Toy 1",
        "category": "Action Figures",
        "price": 19.99
    },
    {
        "_id": "2",
        "name": "Toy 2",
        "category": "Dolls",
        "price": 14.99
    }
]
