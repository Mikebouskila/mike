Description: Returns the number of documents optional limit query parameter.
Query Parameter Description Default
count Returns the number of Toys and pages for a specific limit 5
Respone Example of GET: "/toys/count" 
{
    "count": 25,
    "limit": 5
}

Description: Add a new toy.

Request Body: Toy data in the following format: 

{
    "name": string (2-50 chars),
    "info": string (2-400 chars),
    "category": string (2-50 chars),
    "price": number (0-99999),
    "imgUrl": string (2-400 chars) optional
}
Respone Example of POST: "/toys"
{
    "_id": "...",
    "name": "Action Figure",
    "info": "Poseable action figure with accessories.",
    "category": "Action Figures",
    "imgUrl": "action_figure.png",
    "price": 15,
    "userId": "...",
    "createdAt": "...",
    "updatedAt": "..."",
}

Description: Update an existing toy by it's id.
Params Description Required
id The document's id True

Request Body: Toy data in the following format: 

{
    "name": string (2-50 chars),
    "info": string (2-400 chars),
    "category": string (2-50 chars),
    "price": number (0-99999),
    "imgUrl": string (2-400 chars) optional
}
Respone Example of PUT: "/toys/:id" 
{
    "acknowledged": true,
    "modifiedCount": 1,
    "upsertedId": null,
    "upsertedCount": 0,
    "matchedCount": 1
}

Description: Delete an existing toy by it's id.
Params Description Required
id The document's id True
Respone Example of DELETE: /toys/:id 
{
    "acknowledged": true,
    "deletedCount": 1
}

Endpoint: /users

Description: Add a new user.

Request Body: User data in the following format: 

{
    "name": string (2-150 chars),
    "email": string (2-150 chars),
    "password": string (2-16 chars),
}
Respone Example of POST: "/users"
{
    "_id": "...",
    "name": "netanel",
    "email": "123@gmail.com",
    "password": "***",
    "role": "user",
    "createdAt": "...",
    "updatedAt": "...",
}

Description: Register to an existing user.

Request Body: User data in the following format: 

{
    "email": string (2-150 chars),
    "password": string (2-16 chars),
}
Respone Example of POST: "/users/login" 
{
    "token": "x4...23zx"
}

Description: Get user info from token data.
Respone Example of GET: "/users/login"
